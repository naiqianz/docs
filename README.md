# Product documentation for Uptrace

## Getting started

This project requires pnpm with [shamefully-hoist](https://pnpm.io/npmrc#shamefully-hoist):

```ini
shamefully-hoist=true
```

Install dependencies:

```shell
pnpm install
```

Serve the site:

```shell
pnpm docs:dev
```

Then open http://localhost:8080/get/open-source-apm.html
